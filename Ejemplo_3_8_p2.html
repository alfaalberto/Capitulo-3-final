Definición del Problema

Transformación de una imagen 64×64 para que su histograma adopte una distribución específica.

Valores Iniciales

s0 = 1; s1 = 3; s2 = 5; s3 = 6;
s4 = 6; s5 = 7; s6 = 7; s7 = 7

G(z0) = 0.00 → 0    G(z4) = 2.45 → 2

G(z2) = 0.00 → 0    G(z6) = 5.95 → 6
G(z3) = 1.05 → 1    G(z7) = 7.00 → 7
